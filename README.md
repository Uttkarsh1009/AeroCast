# 🌤️ AeroCast

AeroCast is a modern weather dashboard built with React, Vite, and Express.js. It provides real-time weather information, location search, and multi-day forecasts using the Open-Meteo Weather API and Open-Meteo Geocoding API.

This project was created as a personal project for learning, experimentation, and portfolio demonstration purposes.

## 📖 About

AeroCast allows users to search for cities around the world and instantly view current weather conditions and forecast data through a clean and responsive interface.

The application dynamically adapts weather visuals based on current conditions, creating a more engaging user experience.

---

## ✨ Features

* Search weather by city name
* Real-time weather conditions
* 5-day weather forecast
* Dynamic weather-based backgrounds
* Temperature display
* Wind speed information
* Wind direction information
* Responsive design
* Fast and lightweight user experience
* Error handling for invalid locations

* Current weather
* Temperature
* Wind speed

* Forecast data

### Open-Meteo Geocoding API

Provides:

* City search
* Latitude and longitude lookup
* Location suggestions
